# Synthesis Derivatives

This folder separates synthesis-derived artifacts from the base extraction matrix.

## Rationale

- `04_extraccion/` preserves the manual and traceable extraction matrix used as the methodological base.
- `05_synthesis_derivatives/` contains reproducible analytical transformations created to support article synthesis.

## Contents

- `01_enriched_dataset/articles_data_extraction_enriched.csv`
  Enriched matrix with derived columns for synthesis and comparability.
- `02_normalization/d2/`
  Pre/post counts, mapping tables, and variant notes used to normalize D2.
- `02_normalization/d3/`
  Pre/post counts, mapping tables, and variant notes used to normalize D3.

## Interpretation rule

The columns and tables in this folder do not replace the base extraction matrix. They are derived, reproducible artifacts.
